DNS - это система доменных имён, которая обеспечивает связь названий сайтов с их IP-адресами. Она нужна для конвертации URL сайта в его IP-адрес и наоборот
HTTP - Протокол гипертекствой разметки, используемый для передачи данных в интернете. 
Решает задачу обмена данными между пользовательским приложением/браузером и веб-сервером. Данные пердаются в открытомвиде.
HTTPS - расширение протокола HTTP которое поддерживает технологию шифрования.
версии HTTP:
0.9 самая первая, была создана в начале 90-ых годов (реализована семантика и синтаксис)
1.0 версия протокола является основой современной версии
1.1 версия протокола позволяет клиенту обрабатывать только один запрос в конкретный момент времени
2.0 версия протокола поддерживает выполнение множества запросов через одно соединение одновременно
3.0 протокол будущего
